Projekt Adrian Misiak (73737) i Michał Moskal (73800) - API szyfrujące zdjęcie

W celu uruchomienia programu, uruchom index.py następnie w przeglądarce wejdz na strone localhost:5000

W celu zaszyfrowania zdjęcia wybierz zdjęcie i wyślij. Zaszyfrowany plik znajdzie się w \api_projekt\encrypted_files

W celu odszyfrowania, użyj nazwa_zaszyfrowanego_pliku.enc z powyższego folderu

Kliknij Pobierz plik, plix .txt nie jest wymagany do pobrania.

Odszyfrowany plik znajdzie się w folderze \api_projekt\decrypted_files
